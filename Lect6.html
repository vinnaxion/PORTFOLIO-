Lecture 6 – Multimedia and Forms

Multimedia in HTML

HTML supports multimedia elements like audio and video using built-in tags.

Audio Example

The <audio> tag is used to embed sound in a web page.

Video Example

The <video> tag is used to embed video in a web page.

Embedded YouTube Video (iframe)

The <iframe> tag can embed external content like YouTube videos.

HTML Forms

Forms are used to collect input from users. They use the <form> tag.
