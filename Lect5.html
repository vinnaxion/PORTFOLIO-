What is CSS?

CSS stands for Cascading Style Sheets. It is used to control the appearance and layout of HTML elements on a web page.

Types of CSS

1. Inline CSS

Applied directly to an HTML element using the style attribute.

2. Internal CSS

Written inside a <style> tag in the <head> section of the HTML file.

3. External CSS

Written in a separate .css file and linked using the <link> tag.

CSS Selectors

Element Selector

Targets all elements of a specific type. Example: p { color: red; }

Class Selector

Targets elements with a specific class. Example: .center { text-align: center; }

ID Selector

Targets a single element with a specific id. Example: #para1 { font-weight: bold; }
